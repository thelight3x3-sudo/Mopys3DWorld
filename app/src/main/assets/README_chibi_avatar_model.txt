Beautiful 3D chibi avatar package

This package contains TWO 3D versions:

1) Exact-style extruded 3D model
   - chibi_avatar_exact_style_extruded_3d_model.glb
   - This is a real 3D mesh with thickness and the original design as a texture.
   - Best when you want the model to look closest to the provided image.

2) Volumetric sculpted 3D model
   - chibi_avatar_beautiful_3d_model.glb
   - This is a procedural full 3D chibi sculpt with separate modeled head/body/eyes/hair/beard/fists.
   - Best when you need a fully volumetric game-style base mesh.

Extra formats:
- OBJ + MTL + PNG texture for the exact-style model
- OBJ, PLY, STL for the volumetric sculpted model

Notes:
- Z is up.
- Character front faces negative Y.
- GLB is recommended for Blender, Godot, Unity, Android/OpenGL loaders.
